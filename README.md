# H2O
A complete open source e-commerce solution.

## Install jdk
```bash
curl -s "https://get.sdkman.io" | zsh
sdk install java
sdk install gradle
```

## Build

```bash
git clone https://github.com/kapmahc/h2o.git
gradle build
```

- [sdkman](http://sdkman.io/usage.html)
- [spring](https://spring.io/guides)
