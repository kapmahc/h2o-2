# H2O

A complete open source e-commerce solution by rust language.

## Install rust

```bash
sudo pacman -S rustup
rustup default nightly
rustup update && cargo update # upgrade
```

## Build

```bash
git clone https://github.com/kapmahc/h2o.git
cd h2o
cargo build --release
```
